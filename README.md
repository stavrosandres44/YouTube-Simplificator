# What's YouTube simplificator?
YouTube simplificator is a Chrome Extension that executes on (youtube.com)
to delete HTML classes and simplify the interface

# How does it work?
YouTube simplificator works by having a mutation observer and node vigillance
to monitor YouTube's html to search the HTML classes that are blacklisted 
and delete them instantly and in background.

# Frequently Asked Questions

1. Is YouTube Simplificator Malicious?

————No YouTube simplificator is not malicious, it helps delete HTML elements (mostly ui elements
the average youtube user doesn't use) using mutation observer and node vigillance.

2. How to use it?
- Download the repository
- Extract the zip's contents
- Move the contents to a folder
- Go to `chrome://extensions/`
- Enable developer mode
- Select Load extension without packing
- Click the folder that contains the zip's contents
- Done
